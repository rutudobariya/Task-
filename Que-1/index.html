What is Html Attributes with two example ?



Ans:

HTML attributes provide additional information about HTML elements and are always included in the opening tag of an element. Attributes consist of a name and a value, separated by an equals sign (=). Here are two examples of HTML attributes:


  • src attribute in the < img> element:

    --> This attribute specifies the source (URL) of the image.
    -->In this example, the src attribute is used to define the source file (in this case, "example.jpg") for the image. The alt attribute provides alternative text for the image, which is displayed if the image cannot be loaded.

  • href attribute in the < a> (anchor) element:

    --> This attribute This attribute defines the hyperlink reference.
    -->The href attribute in this example contains the URL ("https://www.example.com") to which the user will be directed when they click the link. The text between the opening and closing tags ("Visit Example.com") is the clickable text that appears on the webpage.
