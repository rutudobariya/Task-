Define list types in html?



Ans:

HTML provides several types of lists to organize and structure content on a webpage. The three main types of lists in HTML are:

1.Ordered List(< ol>):


-->An ordered list is used to create a list where each list item is numbered. The numbering is automatically generated by the browser.
  1. First item
  2. Second item
  3. Third item

2.Unordered List (< ul>):

An unordered list is used to create a bulleted list where each list item is preceded by a bullet or other marker.

  • Red
  • Green
  • Blue

3.Definition List (< dl>):

A definition list is used to group terms (names) with their corresponding definitions.
