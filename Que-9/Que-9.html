Define Z-index ? create like this example using z-index



Ans:

when you have multiple elements that overlap on a webpage, the z-index property determines which element appears on top of the others. The element with a higher z-index value will be visually stacked above the elements with lower z-index values.

In this example, element1 will be displayed above item2 because it has a higher z-index value. If two elements have the same z-index, the one that appears later in the HTML source order will be displayed on top
