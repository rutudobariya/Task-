What Are void elements in html ? explain with 5 void elements ?



Ans:

=> Void elements, also known as self-closing or empty elements, are HTML elements that do not have a closing tag. They are used to embed media, break lines, or provide other single-purpose functionality. Void elements in HTML are written in a self-closing form, usually with a trailing slash before the closing angle bracket. Here are five examples of void elements:



  1. < img> - Image Embedding:

    The < img> element is used to embed images on a webpage.

  2. < br> - Line Break:

    The `< br>` element is used to create a line break within text.

    This is some text.
    And this is on a new line.

  3. < input> - Form Input::

    The < input> element is used to create various types of form inputs..

  4. < hr> - Horizontal Rule:

    The< hr> element is used to create a horizontal rule (line) to separate content..

    This is a paragraph.

    This is another paragraph.

  5. < meta> - Metadata:

    The element is used to provide metadata about the HTML document, such as character encoding, viewport settings, etc.

    < meta charset="UTF-8" />
