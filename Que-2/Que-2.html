What is Marquee tag in html ? Explain with two example ?



Ans:

  1. The < marquee> tag is an HTML element that was traditionally used to create a scrolling text or image effect on a webpage. It's important to note that the < marquee> tag is considered deprecated in HTML5, and it's generally not recommended to use it in modern web development due to accessibility and usability concerns. Instead, CSS and JavaScript are preferred for achieving similar effects..
